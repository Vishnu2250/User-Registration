# PHP User Authentication System 🔐

This project is a simple and secure User Authentication System developed using PHP, MySQL, HTML, CSS, and Bootstrap. The application allows users to register, log in, and access a personalized dashboard using session-based authentication.

The project includes a responsive landing page with Sign Up and Login options, making it easy for users to access the system through a clean and modern interface.

## ✨ Features

* User Registration System
* User Login Authentication
* Password Hashing using PHP
* Session Management
* Secure Login Validation
* Responsive Landing Page
* Bootstrap User Interface
* Personalized Dashboard Access
* Error Handling and Validation

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap
* XAMPP/WAMP Server
